== README

ABOUT:

  This is a Micro-blogging-App that allows Users to Post their blog.
  
TECHNOLOGIES USED:

  Front-end:Bootstrap-Saas,CSS
  Back-end:Ruby,Ruby on Rails,Ruby gems include-Devise.
  
  FEATURES:
  
    Users Signup, create a Post, Edit and Delete a Post.
    Users can also  Comment on a Post.
